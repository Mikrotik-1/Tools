🚀 احصل على Chat ID لتفعيل إشعارات Telegram

1 افتح Telegram

اذهب إلى تطبيق Telegram على هاتفك أو الكمبيوتر

افتح Get IDs Bot
2 ابدأ المحادثة مع البوت

بعد فتح الرابط، اضغط على زر START أو أرسل:

/start
3 احصل على Chat ID

سيرسل لك البوت رسالة تحتوي على رقم مثل:

123456789
💡 ملاحظة: هذا الرقم خاص بك فقط ولا تشاركه مع أحد
